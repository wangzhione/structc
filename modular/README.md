## 0. 存在合理性

    a). 条理清晰的让 C 去提供服务, 减少开发成本

## 1. linux 部署

    推荐采用 Best New Ubuntu x64 GCC 环境搭建

    a) 编译设置

        -lm -lpthread

    b). 单元测试规则

        {接口}.h -> test :: {接口}_test.c ->

            {接口}_test(void) {
                ...
                ..
                .
            }

        测试文件. 方便统一生成特定内容.
